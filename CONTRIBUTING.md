# Contributing

Refer to the [Ansible community guide](https://docs.ansible.com/ansible/devel/community/index.html) for general instructions  on how to contribute.

Refer to the [Community OpenWrt Module Developer Guide](https://ansible-collections.github.io/community.openwrt/branch/main/docsite/mod_dev_guide.html)
for specific instructions for developing modules for community.openwrt.

And you might also want to check the full
[community.openwrt collection documentation site](https://ansible-collections.github.io/community.openwrt/branch/main/).
