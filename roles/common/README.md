# Ansible Role: community.openwrt.common

## Description

This role contains the common handlers for the community.openwrt collection.
If the user, for any reason, decide not use the initialization role
`community.openwrt.init`, they may import this role and use those handlers.

See also [Community.OpenWrt Collection Docs](https://ansible-collections.github.io/community.openwrt/branch/main/index.html) for more information.

**Note:** The recommended way of using `community.openwrt` is to use the
initialization role.
