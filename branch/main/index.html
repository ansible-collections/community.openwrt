Description

The community.openwrt collection provides Ansible code to manage OpenWrt devices. Supports OpenWRT devices without Python installed by using shell-based modules.

Author:

  • Alexei Znamensky (github.com/russoz)

Supported ansible-core versions:

  • 2.17.0 or newer
